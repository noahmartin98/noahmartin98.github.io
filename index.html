Algorithms and Data Structures

In the area of algorithms and data structures, I focused on adding features to my application and improve the efficiency of my code. One of the features that I added is the ability to switch between different season statistics and total statistics on the leaders pages. I accomplished this by adding a select element that reloads the page when changed, and using a GET statement to retrieve the selected value and update the table with a new query. I also added the ability to sort the tables and search for players using a javascript table library called DataTables. After linking to the library’s files, I just needed to make a few changes and write a small amount of javascipt to get the table sortable exactly how I wanted. Also, I added a new page that displays playoff brackets for the seasons in the database. To improve the efficiency of my project, I looked back through my code, removed unnecessary code, and simplified complex code. Originally, each php file had code at the top that connected it to the database. To simplify the code, I created a databaseConnect.php file which I required at the top of each page, reducing the amount of code needed. Also, this made updating my database information when I switched to a code database much less tedious, as I only needed to update one file. I also simplified many of my sql queries that were unnecessarily complicated. The processes of adding new features and improving the efficiency of my code allowed me to work on my programming using standard coding practices and code reviewing skills.
